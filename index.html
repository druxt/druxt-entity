DruxtJS Entity

Drupal Display Mode powered Entity, Form and Field Vue.js component system.

Get started

The DruxtEntity module provides a Drupal (opens new window) Display Mode powered Entity, Form and Field component system for your Nuxt.js (opens new window) application.

# DruxtEntity

The DruxtEntity component renders content entities using Drupal's View display modes.

<DruxtEntity :type="resourceType" :uuid="uuid" :mode="displayMode" />
1

Get started with the Guide and API Documentation.

# DruxtEntityForm

The DruxtEntityForm component uses Drupal's Form displays modes for content creation and editing.

<DruxtEntityForm :type="resourceType" :mode="displayMode" />
1

Get started with the Guide and API Documentation.

# DruxtJS

DruxtJS is a framework to connect a Drupal (opens new window) JSON:API backend to a Nuxt.js (opens new window) application.

Find out more at https://druxtjs.org (opens new window)
