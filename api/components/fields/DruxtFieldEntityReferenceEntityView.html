# DruxtFieldEntityReferenceEntityView

Entity Reference Entity View field.

  • Renders the reference entity with the <druxt-entity /> component.

This component is intended to be rendered by the <DruxtField /> component.

See

  • DruxtEntity
  • DruxtField

Example

<DruxtField
  :data="{
    data: {
      id: '88d46b5c-07c8-4d80-8f4d-e45ff2639d56',
      type: 'media--image'
    }
  }"
  :schema="{
    id: 'field_media_image',
    type: 'entity_reference_entity_view',
    settings: {
      display: {
        link: false,
        view_mode: 'medium_8_7'
      }
    }
  }"
/>
