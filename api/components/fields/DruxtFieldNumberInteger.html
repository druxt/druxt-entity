# DruxtFieldNumberInteger

Number Integer field.

This component is intended to be rendered by the <DruxtField /> component.

See: DruxtField
Example

<DruxtField
  data="5"
  :schema="{
    id: 'field_preparation_time',
    settings: {
      config: {
        prefix: '',
        suffix: ' minutes'
      },
      display: {
        prefix_suffix: true
      }
    },
    type: 'number_integer'
  }"
/>
