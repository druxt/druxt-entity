# DruxtEntityMixin

Provides Vue.js properties to render a DruxtEntity or DruxtEntityForm Wrapper component.

Kind: global mixin
Example

<script>
import { DruxtEntityMixin } from 'druxt-entity'

export default {
  mixins: [DruxtEntityMixin],
}
</script>
