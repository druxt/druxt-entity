# DruxtEntityNuxtModule()

Nuxt module function to install Druxt Entity.

  • Adds dependant modules.
  • Adds Nuxt plugin.
  • Adds Nuxt Storybook integration.

The module function should not be used directly, but rather installed via your Nuxt configuration file.

Options are set on the root level druxt Nuxt config object.

Kind: global function

Properties

Name Type Description
options.druxt object DruxtJS root level options.
options.druxt.baseUrl string Base URL of Drupal JSON:API backend.

Example

// `nuxt.config.js`
module.exports = {
  modules: [
    'druxt-entity'
  ],
  druxt: {
    baseUrl: 'https://demo-api.druxtjs.org'
  }
}
